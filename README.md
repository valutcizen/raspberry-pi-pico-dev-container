# Raspberry Pi Pico Dev Container

Lightweight Raspberry Pi Pico Dev Container. Based on [raspberry-pi-pico-docker-sdk](https://github.com/lukstep/raspberry-pi-pico-docker-sdk) by [lukstep](https://github.com/lukstep)

# Usage

## First run

Clone or fork&clone repository. Open it in VS Code with Dev Containers extension installed. Press `F1` and choose `Dev Containers: Reopen in Container`. Wait for re-open and container initialization.

## First build

Press `F1`, choose `Tasks: Run Build Task` and choose `cmake & make`.

## Second and later builds
Press `F1`, choose `Tasks: Run Build Task` and choose `make build`.
